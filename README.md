# Shieldher
Shieldher is a private incident documentation platform that hepls indivisuals securely record and preserve evidence of harassment or unsafe events.

The platform allows users to:
-Record incident details(date,time,location,description)
-Attach evidence(images,audio,video,PDF)
-Store records privetely
-Access records later if they decide to take action

The goal is to help victims document early evidence safely before formal reporting.
